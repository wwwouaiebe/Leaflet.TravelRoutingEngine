pointsDistance(latLngStartPoint, latLngEndPoint)

This function returns the distance between two points Since v1.7.0 we use the simple spherical law of cosines formula (cos c = cos a cos b + sin a sin b cos C). The delta with the Leaflet function is always < 10e-3 m. The error due to the earth radius is a lot bigger. Notice: leaflet uses the haversine formula.
